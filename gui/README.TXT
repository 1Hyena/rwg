Random World Generator (Graphical User Interface)
made by Erich Erstu 2012

Source code: https://github.com/1Hyena/rwg


KEYS:
--------------------------------------------------------------------------
ESC          - Exit the program.
F11          - Stop/Continue redrawing the map.
F12          - Export the current map to PNG format heightmap and biome textures.
SPACE        - Pause/Continue the generation process.
PAGE UP      - Display next layer.
PAGE DOWN    - Display previous layer.
ARROW KEYS   - Move the map.
NUMPAD PLUS  - Zoom in.
NUMPAD MINUS - Zoom out.
